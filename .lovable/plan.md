

# BockLock - Professional B2B Security Website

## Project Overview
A premium, multi-language corporate website for BockLock, a B2B security company specializing in high-security locks and access control systems for banks and financial institutions.

---

## Design System

### Brand Colors
- **Primary Navy:** #001F3F (deep trust, security)
- **Steel Gray:** #4A4A4A (professional, industrial)
- **Pure White:** #FFFFFF (clean, spacious)
- **Gold Accent:** #D4AF37 (premium, trust)
- **Success Green:** #00A676 (trust indicators)

### Typography
- **Headings:** Montserrat Bold (strong, commanding)
- **Body:** Inter (clean, readable, professional)

### Design Principles
- Clean minimal layout with generous whitespace
- Subtle shadows and smooth hover effects
- Security-themed iconography (shields, locks, vaults)
- Mobile-first responsive design
- Dark mode support with toggle

---

## Internationalization System

### URL Structure
- `/en/` - English (default, redirects from `/`)
- `/de/` - German
- `/sr/` - Serbian

### Language Switcher
- Elegant dropdown in header with country flags
- Preserves current page when switching languages
- Stores preference in localStorage

### Content
- All text content translated across all three languages
- Professional banking/security terminology
- Proper Serbian Cyrillic or Latin script support
